# Student Management System

A console-based Student Management System built in Python that allows 
you to manage student records with persistent storage using JSON.

## Features

- Add new students with multiple subject marks
- View all students and their marks
- Search for a specific student
- Delete a student record
- Find the class topper
- Calculate class average
- View pass/fail list
- Data persists between runs using JSON file storage
